    Team 5450

    This project contains the source code for FRC team 5450's 2015 robot. The project was written and compiled with the WPILib packages in Eclipse, with the language being c++. As for the code, the annotated version will be available soon on a different page.

    About

    The autonomous program:

    • Overall, there are 5 different PID subsystems, (custom, not prebuilt WPILib) 4 of which are in the autonomous program
    • The robot is using custom voids to control exactly what it is doing. That way, everything is clean and ordered, and multiple arguments can be required without a mess.

    The tele-operated program

    • The program uses a PID loop to change the motor compensation depending on how many crates are on the elevator
    • The program was optimized to ensure the least amount of button usage. (ensures ease of control under pressure)
    • Features: tank drive and special axis controls
